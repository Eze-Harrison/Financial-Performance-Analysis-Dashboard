# Financial-Performance-Analysis-Dashboard

# Project Overview

In this project, analyzed financial transaction data to uncover revenue trends, profitability insights, customer performance, and business growth opportunities.The dashboard was created using Power BI to transform raw financial data into interactive business intelligence insights.

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization
- DAX

## Dataset
The dataset contains financial transaction records including:

- Transaction details
- Customer information
- Product details
- Revenue
- Cost
- Profit
- Sales channels
- Regional performance

## Business Questions Answered

1. What is the total revenue generated?
2. What is the overall gross profit and profit margin?
3. Which product category generates the highest revenue?
4. Which products contribute the most profit?
5. Which region performs best?
6. How does revenue change monthly?
7. Which customer segment generates the most value?
8. Which sales channel performs best?
9. What products should the business prioritize?

## Key KPIs

- Total Revenue
- Total Cost
- Gross Profit
- Quantity Sold
- Revenue by Category
- Profit by Product
- Regional Sales Performance

## Insights

- Electronics generated the highest revenue.
- Laptop products contributed the highest profit.
- Sales performance varies across regions and channels.
